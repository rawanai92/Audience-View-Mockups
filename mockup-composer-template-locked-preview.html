What's locked, what's not

  • Template name + Approved + Locked pill at the top of the inserted card make it obvious the content is governed by the Library.
  • Body, image, and buttons render exactly as the customer will receive them, on the WhatsApp chat background. Variables show the contact's resolved values.
  • Button types appear as small tags (URL · Quick reply · Phone · Copy code) so the agent can tell what each one does.
  • Replace opens the template picker (same as the original insert flow).
  • Remove clears the inserted template, returns to an empty composer.
  • Send sends the rendered template. Disabled if any variable can't resolve for the current contact; error message identifies which.
